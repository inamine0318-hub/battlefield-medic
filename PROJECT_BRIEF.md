## 🌍 世界観・背景設定

### 舞台
**第二次世界大戦、太平洋戦線の島嶼戦闘地**

戦地の最前線。砲撃が響き渡り、機関銃掃射が絶え間なく続く。地雷原と有刺鉄線が張り巡らされた危険地帯。

### プレイヤーの立場
**衛生兵（メディック）**

銃を持たない医療従事者。敵を倒すのではなく、味方の負傷兵を救うのが職務。

### 映画的インスピレーション
この作品は映画『**ハクソー・リッジ**（Hacksaw Ridge）』にインスパイアされている。

- 主人公デズモンド・ドス：銃を持たずに、絶望的な戦場で百人以上の兵士の命を救った実在の衛生兵
- 「武器を持たず、人を助ける」という普遍的なテーマ
- 戦闘の恐怖と、人命救助の使命感のジレンマ

**このゲームは、衛生兵の葛藤と勇気を体験するもの。**

## 🎮 ゲームの面白さ

### コアコンセプト
**「敵を倒すゲーム」ではなく「人命を救うゲーム」**

一般的なシューティングやアクションゲームは「敵撃破」が目的。しかし衛生兵の使命は**敵を倒すことではなく、負傷兵を救うこと**。

### プレイ体験の流れ

```
【ゲーム開始】

STAGE 1: 砲撃が降り注ぐ海岸
┌─────────────────────────────────────┐
│  キャラクターが自動で前に走り続ける   │
│  砲撃が空から降ってくる              │
│  タップしてジャンプして避ける         │
│  走りながら負傷兵を見つけて接触       │
│  ✓ 負傷兵救助 → スコア100点          │
│  ✓ ブーツ取得 → 二段ジャンプ可能     │
│  ⚠ ダメージ受けた → HP-1, ブーツ喪失 │
└─────────────────────────────────────┘
  ↓（残り時間まで繰り返し）
【STAGE 1 クリア】

STAGE 2: 塹壕地帯の激戦
┌─────────────────────────────────────┐
│ 砲撃 + 機銃掃射の複合攻撃            │
│ より難しい回避パターン               │
│ ブーツの重要性が増す                 │
│ より多くの負傷兵が出現               │
└─────────────────────────────────────┘
  ↓
STAGE 3: 廃墟の村での最後の救助活動
┌─────────────────────────────────────┐
│ すべてのギミックが登場               │
│ 制限時間が最短（260秒）              │
│ 絶望的な状況の中での人命救助          │
│ 最高のスコアを目指す                 │
└─────────────────────────────────────┘
  ↓
【MISSION COMPLETE または GAME OVER】
```

### なぜ面白いのか

1. **「避ける」と「救う」の二重構造**
   - 一般的なランゲームは「避けるだけ」
   - このゲームは「避けながら同時に誰かを助ける」
   - → 時間と危険のトレードオフ
   - → 「あと1人だけ助けたい」という衝動

2. **時間制限による緊張感**
   - 単なる生き残りではなく、「限られた時間で最大何人救えるか」
   - 残り時間が減ると焦りが増す
   - スコアを諦めて安全を取るか、危険を冒すか

3. **二段ジャンプの獲得と喪失**
   - ブーツ取得で戦術の幅が広がる
   - ダメージで失うので、常に「このジャンプは必要か」と判断
   - → アイテム管理の戦略性

4. **レトロ感による心理的距離感**
   - ピクセルアートの懐かしさ
   - 80年代アーケードゲームの純粋さ
   - 現代的リアリズムではなく「象徴的な表現」だからこそ、戦争の悲劇をストレートに感じられる
   - → プレイヤーが感情移入しやすい

5. **短時間で完結**
   - 1セッション5分程度でクリア可能
   - モバイルゲームとしてのニーズ合致
   - でも高難易度で繰り返しプレイしたくなる

## 🎨 ビジュアル・世界観の表現

### ドット絵の美学

**8×12ピクセルの衛生兵**
```
    走ってる
    肉体感
    人間らしさ
    ↓
  [走行アニメーション]
     (3フレーム)
    ↓
プレイヤーが「キャラクターを操作している」という感覚が生まれる
```

キャラクターは単なる「点」ではなく、**走っている人間**として描かれている。
- IDLE：立っている状態
- RUN1, RUN2：走行サイクル（足の動き）

### 赤十字マーク
- 医療兵であることの**視覚的シンボル**
- 「敵を倒す兵士」ではなく「人を救う医療者」という身分を表現
- ピクセルアートの中で最も目立つ要素

### 障害物のビジュアル表現

**砲撃爆発**
- 3段階の色層：外側オレンジ → 中央黄 → 中心黄
- 爆発の「厚み」を表現
- スピード感のある脅威

**機銃掃射**
- 画面全幅の黄色い帯
- 避けられない感覚＝絶望感
- 「逃げ場がない」という恐怖を視覚化

**有刺鉄線**
- マゼンタ色（紫）
- 地面に張られた脅威
- 「地面では安全ではない」という認識

### 負傷兵のビジュアル
- 肌色の頭 + 赤い体 + 黒い脚
- 人間の身体をミニマルに表現
- プレイヤーが「人間を救っている」という実感

### 背景演出
- スクロール背景：戦場の廃墟感
- 砲撃跡（薄暗いグレー）
- 血痕表現（赤い半透明のブロック）

→ **戦場というシリアスな世界観を、ピクセルアートで洗練して表現**

## 🎯 ゲームシステム詳細

### 基本操作
- **ジャンプ**: 画面をタップ
- **二段ジャンプ**: ブーツ取得時、空中で再度タップ

シンプル。複雑な操作は一切ない。

### ライフ・ダメージシステム

```
初期状態
HP: ●●●●● (5)

砲撃に当たった
HP: ●●●● (4) + ブーツ喪失

再度当たった
HP: ●●● (3)

3回当たった
HP: ●●●●● (0)
↓
GAME OVER
```

**仕様**:
- 初期HP: 5
- ダメージ: -1 HP
- ブーツ喪失: ダメージ時に同時喪失
- 無敵時間: 30フレーム（ダメージ後、点滅状態）

### アイテム: ブーツ

**何か**:
- 二段ジャンプ可能にする装備

**出現**:
- 280フレーム毎（約4.7秒毎）
- 50%確率で出現
- ランダムX座標、スクロール奥から出現

**効果**:
- 地上ジャンプ: 11パワー
- 空中ジャンプ: 11パワー（ブーツ装備時のみ）

**喪失**:
- ダメージを受けると即座に喪失
- 複数取得可能（何度も出現して取得できる）

**戦略**:
- ブーツなし: 砲撃は飛び越えられるが、複雑な配置には対応不可
- ブーツあり: より高く、より複雑な回避が可能

### 負傷兵システム

**出現パターン**:
- ステージ1: 135フレーム毎（約2.25秒毎）
- ステージ2: 120フレーム毎（2秒毎）
- ステージ3: 105フレーム毎（約1.75秒毎）

**外見**:
- 肌色の頭 + 赤い体 + 黒い脚（20×20px）

**救助方法**:
- 接触すると自動救助
- 青緑色のパーティクル発生 + 成功音（800Hz）

**スコア**:
- 1人: 100点
- 最大: 15～20人（全ステージで計算すると約1500点）

**プレイヤーの心理**:
- 「あと1人だけ救いたい」という欲望
- スコア狙いプレイ（ハイスコア更新）
- 「時間が足りない」という焦燥感

### 敵（ギミック）システム

#### ステージ1: 砲撃のみ
**砲撃爆発**
- 出現: 70フレーム毎、60%確率
- サイズ: 50×50px
- ビジュアル: 3段階の色（外側オレンジ → 黄 → 黄）
- 回避方法: タップしてジャンプで躱す
- 難易度: ★☆☆（初級）

#### ステージ2: 砲撃 + 機銃掃射
**砲撃爆発** (ステージ1と同じ)

**機銃掃射**
- 出現: 70フレーム毎、20%確率
- サイズ: 360×20px（画面全幅）
- ビジュアル: 黄色 + オレンジ条
- 回避方法: 高さを変えて躱す（より複雑）
- 難易度: ★★☆（中級）

#### ステージ3: 全ギミック
**砲撃爆発**, **機銃掃射**に加えて：

**有刺鉄線**
- 出現: 70フレーム毎、20%確率
- サイズ: 40×40px
- ビジュアル: マゼンタ（紫）
- 配置: 地面に張られている
- 回避方法: ジャンプで飛び越す
- 難易度: ★★★（上級・複合ギミック）

### ステージ進行と難易度

```
STAGE 1 (300秒)
├─ 砲撃のみ
├─ 負傷兵出現頻度：低
└─ 難易度：★☆☆ (チュートリアル的)

STAGE 2 (280秒)
├─ 砲撃 + 機銃掃射
├─ 負傷兵出現頻度：中
└─ 難易度：★★☆ (中級)

STAGE 3 (260秒)
├─ 砲撃 + 機銃掃射 + 有刺鉄線
├─ 負傷兵出現頻度：高
└─ 難易度：★★★ (上級・最終試験)
```

## 🎵 サウンドデザイン

### 効果音

**通常ジャンプ**: 400Hz × 0.1秒
- 軽い上昇音
- プレイヤーが地面から飛び上がる感覚

**二段ジャンプ**: 500Hz × 0.1秒
- より高い周波数

- 低い爆発音

**負傷兵救助**: 800Hz × 0.3秒
- 高い成功音

## 🎨 デザイン哲学

### なぜピクセルアート？

1. **普遍性**: 年代を超えた懐かしさと新しさ
2. **シンプリシティ**: 余計な装飾がなく、本質的な表現
3. **制約による創意**: 限られたピクセルで最大限の表現力
4. **感情移入**: リアルではなく「象徴的」だからこそ共感しやすい
5. **パフォーマンス**: ブラウザでも軽く動作

### レトログリーン色のUI

- 1980年代アーケードゲームのオマージュ
- プレイヤーに「懐かしい」という感情で安心感を与える
- 戦争という重いテーマと、軽快なゲーム体験のバランス

### 「SAVE LIVES. NOT KILL ENEMIES.」

このゲーム全体を表現する一文。

通常のゲームは「敵を倒す」
このゲームは「人を救う」

その違いが全てを変える。

4. **インディーゲーム的価値**
   - 小さいが深い体験

   - ドット絵の美しさ
   - Web標準技術でのシンプル実装
