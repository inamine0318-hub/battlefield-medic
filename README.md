# 戦場の衛生兵（Battlefield Medic）

## 📌 ゲーム概要

**タイトル**: 戦場の衛生兵  
**英名**: Battlefield Medic  
**ジャンル**: スマートフォン向け縦画面ラン&回避アクション  
**プラットフォーム**: Web（HTML5 Canvas）  
**対象プレイ時間**: 1セッション3～5分  
**制作者**: Claude Haiku 4.5  
**公開日**: 2026-06-20

## 🌍 世界観・背景設定

### 舞台
**第二次世界大戦、太平洋戦線の島嶼戦闘地**

砲撃が響き渡り、機関銃掃射が絶え間なく続く。地雷原と有刺鉄線が張り巡らされた最前線の戦地。

### プレイヤーの役割
**医療兵（メディック）**

銃を持たない医療従事者。敵を倒すのではなく、負傷した味方兵を救うのが職務。

### ゲームのテーマ
```
「武器を持たず、人を助ける」

通常のゲーム：敵を倒す → スコア獲得
このゲーム：人命を救う → スコア獲得

その違いが全てを変える。
```

### 映画的インスピレーション
映画『**ハクソー・リッジ**（Hacksaw Ridge）』

実在の医療兵デズモンド・ドスは、銃を持たずに絶望的な戦場で百人以上の兵士の命を救った。このゲームは、そのような勇気と使命感を体験するもの。

## 🎮 ゲームの面白さ

### コアメカニック
**「避けること」と「救うこと」の同時進行**

```
一般的なランゲーム：
障害物を避ける → スコア加算

Battlefield Medic：
障害物を避ける + 負傷兵を救う → スコア加算

→ 時間と危険のトレードオフが生まれる
→ 「あと1人だけ助けたい」という心理
→ 高いリプレイ性
```

### 面白さの要素

**1. 二重のプレッシャー**
- 敵（ギミック）からの脅威：避けないとダメージ
- 時間制限：残り時間でどれだけ救えるか
- → 緊張感が常に存在

**2. アイテムの獲得と喪失**
- ブーツ取得 → 戦術の幅が広がる
- ダメージで喪失 → 「このジャンプは必要か」と常に判断
- → 資源管理の戦略性

**3. スコアとリプレイ**
- 最大スコア：約1500点
- ハイスコア更新への動機
- 総救助人数の累積記録
- → 何度も遊びたくなる設計

**4. レトロ感による感情移入**
- 80年代アーケードゲームの美学
- シンプルなドット絵
- 現実ではなく「象徴的な表現」だからこそ、戦争の悲劇をストレートに感じられる

## 🕹️ 操作方法

| アクション | 入力 | 説明 |
|-----------|------|------|
| **ジャンプ** | 画面をタップ | 地面から上に飛ぶ（通常時） |
| **二段ジャンプ** | ブーツ装備後、空中でタップ | ブーツ取得時のみ可能 |

**シンプル**: タップのみ。複雑な操作は一切ない。

## ❤️ ライフシステム

### HP管理
```
初期状態：HP 5/5

━━━━━

ダメージを受けた：HP 4/5 + ブーツ喪失
↓
━━━━

再度ダメージ：HP 3/5
↓
━━━

3回ダメージ：HP 0/5
↓
GAME OVER
```

### 詳細仕様
- **初期HP**: 5ポイント
- **ダメージ値**: -1 HP（どのギミックも同じ）
- **無敵時間**: ダメージ後30フレーム（点滅状態）
- **HP 0時**: ゲームオーバー

### HP表示
- キャラクター上部に赤いドット（●）で表示
- 1ドット = 1 HP

## 🎁 アイテムシステム

### ブーツ（唯一のアイテム）

#### 外見
- 黄色とオレンジ色の靴型
- キャンバスサイズ：16×16ピクセル
- ステージ内を上から下に流れてくる

#### 出現仕様
```
出現タイミング：280フレーム毎
出現確率：50%
出現位置：ランダムX座標、スクロール奥から出現

280フレーム × 0.5確率 = 平均560フレーム毎

ステージ1 (300秒 = 18000フレーム)
→ 期待出現数：約3～4個

ステージ2 (280秒 = 16800フレーム)
→ 期待出現数：約3～4個

ステージ3 (260秒 = 15600フレーム)
→ 期待出現数：約3～4個
```

#### 取得方法
- キャラクターが接触すると自動取得
- 取得時に黄色いパーティクル発生

#### 効果
**二段ジャンプの解放**

```
ブーツなし：
- 地上ジャンプ：1回（パワー11）
- 空中ジャンプ：不可
- 最大高さ：制限あり

ブーツ装備：
- 地上ジャンプ：1回（パワー11）
- 空中ジャンプ：1回（パワー11）＝計2段ジャンプ
- 最大高さ：より高く到達可能
```

#### 喪失条件
- ダメージを受けると**即座に喪失**
- ブーツ喪失 → 二段ジャンプ不可に

#### 戦略的価値
```
ステージ1：
- ブーツなしでも大半のギミック回避可能
- 「あると便利」レベル

ステージ2：
- 機銃掃射出現で重要性増す
- より複雑な回避パターンに対応

ステージ3：
- ほぼ必須
- 有刺鉄線＋他ギミック組み合わせで二段ジャンプが必要
- ブーツ温存戦略が重要
```

#### 複数取得
- 複数出現すれば何度でも取得可能
- ブーツを失わずに次のブーツを取得できれば、常時二段ジャンプ可能

## ⚠️ 敵・ギミックシステム

### 共通仕様
```
出現方式：スクロール奥から手前に流れてくる
当たり判定：矩形判定（ピクセルパーフェクト）
ダメージ：-1 HP + ブーツ喪失
無敵時間：ダメージ後30フレーム
```

### ギミック1：砲撃爆発

#### 外見
```
層1（外側）: オレンジ色 (#ff6600)
層2（中央）: 黄色     (#ffaa00)
層3（内側）: 黄色     (#ffff00)
```

3段階の色で爆発の厚みを表現

#### スペック
| 項目 | 値 |
|------|-----|
| **サイズ** | 50×50ピクセル |
| **出現条件** | 70フレーム毎 + 60%確率 |
| **移動方向** | 上から下へ自動スクロール |
| **回避方法** | ジャンプで飛び越す |

#### 出現確率計算
```
70フレーム毎に判定
そのうち60%が砲撃爆発
40%が他のギミック（ステージ2以降）

結果：
- ステージ1では砲撃が主流（他ギミックなし）
- ステージ2以降は砲撃+他ギミック混在
```

#### 難易度
- ステージ1：★☆☆（初級・砲撃のみ）
- ステージ2：★★☆（中級・砲撃が基本）
- ステージ3：★★★（上級・砲撃が最多）

---

### ギミック2：機銃掃射

#### 外見
```
帯状（全幅）: 黄色 (#ffff00)
条紋パターン: オレンジ色（8本の帯）

━━━━━━━━━━━━━━━
█ █ █ █ █ █ █ █
━━━━━━━━━━━━━━━
```

画面全幅を占める黄色い帯で横切る

#### スペック
| 項目 | 値 |
|------|-----|
| **サイズ** | 360×20ピクセル（画面全幅） |
| **出現条件** | 70フレーム毎 + 20%確率 |
| **登場ステージ** | ステージ2以降 |
| **移動方向** | 上から下へ自動スクロール |
| **回避方法** | 高さを変えて躱す（砲撃より複雑） |

#### 回避難度
```
砲撃：
「避ける場所がある」→ 比較的簡単

機銃掃射：
「画面全幅を占める」→ より複雑な判断が必要
「上下どちらかに避ける」→ 段差を活用
```

#### 登場パターン
```
ステージ1：出現しない（砲撃のみ）

ステージ2：
- 砲撃60% + 機銃20% + 有刺鉄線0%
- 複合ギミックの登場

ステージ3：
- 砲撃60% + 機銃20% + 有刺鉄線20%
- すべてのギミックが混在
```

---

### ギミック3：有刺鉄線

#### 外見
```
層1（外側）: マゼンタ色  (#ff00ff)
層2（内側）: 濃マゼンタ  (#ff0088)

斜線パターンで張られた状態を表現
```

#### スペック
| 項目 | 値 |
|------|-----|
| **サイズ** | 40×40ピクセル |
| **出現条件** | 70フレーム毎 + 20%確率 |
| **登場ステージ** | ステージ3のみ |
| **配置位置** | 地面に張られている |
| **移動方向** | 上から下へ自動スクロール |
| **回避方法** | ジャンプで飛び越す |

#### 戦略的意味
```
砲撃・機銃：
「上下に移動して躱す」

有刺鉄線：
「地面に張られた脅威」
「地面では安全ではない」という認識
→ 「高くジャンプして超える必要」
→ 「ブーツが重要になる」
```

#### ステージ3の難易度UP要因
- 砲撃（降ってくる）+ 機銃（全幅）+ 有刺鉄線（地面）
- 「上」「中」「下」すべての脅威が存在
- ブーツがないと回避不可能なパターンが多い

## 📊 ステージ構成

### ステージ1：海岸戦線（Coastal Combat）

**テーマ**: 初級チュートリアル的難易度

| 項目 | 値 |
|------|-----|
| **制限時間** | 300秒 |
| **メインギミック** | 砲撃爆発のみ |
| **負傷兵出現** | 135フレーム毎（低頻度） |
| **難易度** | ★☆☆ |
| **期待スコア** | 300～400点 |

**敵出現パターン**:
```
70F毎の判定：
├─ 60% → 砲撃爆発出現
├─ 20% → 出現なし（ステージ1では機銃なし）
└─ 20% → 出現なし（ステージ1では有刺鉄線なし）

結果：砲撃がメイン、他ギミック一切なし
```

**プレイ体験**:
- 「タップしてジャンプ」の基本操作習得
- 「負傷兵を救う」喜びの体験
- ブーツの有用性を学ぶ
- 次ステージへの準備

**テーマ**: 複合ギミックの登場で難易度UP

| 項目 | 値 |
|------|-----|
| **制限時間** | 280秒 |
| **メインギミック** | 砲撃 + 機銃掃射 |

**敵出現パターン**:
```
70F毎の判定：
```

**新要素**:
- 機銃掃射の回避テクニック
- 砲撃と機銃が同時に出現することも

**プレイ体験**:
- ギミック複合による難易度上昇
- より複雑な回避パターン
- 「難しいけど楽しい」という感覚

### ステージ3：廃墟の村（Ruined Village）

**テーマ**: 最終試験・全力プレイ

| 項目 | 値 |
|------|-----|
| **制限時間** | 260秒（最短） |
| **メインギミック** | 砲撃 + 機銃掃射 + 有刺鉄線 |
| **負傷兵出現** | 105フレーム毎（高頻度） |
| **難易度** | ★★★ |
| **期待スコア** | 500～600点 |

**敵出現パターン**:
```
70F毎の判定：
├─ 60% → 砲撃爆発出現
├─ 20% → 機銃掃射出現
└─ 20% → 有刺鉄線出現（新ギミック登場）
```

**最難関の要因**:
- すべてのギミックが混在
- 制限時間が最短（260秒）
- 負傷兵出現頻度が最高
- ブーツ温存戦略が必須
- 「スコアを諦めるか、危険を冒すか」の判断が求められる

**クリア条件**:
- HP 0になるまでに時間制限終了を待つ
- 負傷兵をできるだけ多く救助
- スコア最大化を目指す（ハイスコアアタック）

- **指数減衰**: すべての音は徐々に音量が低下

- **フォールバック**: 非対応ブラウザでは無視（エラーなし）

- **ステージ1-3**: 緊迫した戦場音楽（**将来実装予定**）

## 🌟 ゲーム哲学

### なぜピクセルアート？

1. **普遍性**：年代を超えた懐かしさ
2. **シンプリシティ**：本質的な表現力
3. **感情移入**：リアルではなく「象徴的」だから
4. **パフォーマンス**：ブラウザで軽快に動作

### なぜレトログリーン？

- 1980年代アーケードゲームへのオマージュ
- 重いテーマ（戦争）と軽快なゲーム性のバランス

### 「SAVE LIVES. NOT KILL ENEMIES.」

このゲーム全体を表現する一句。

```
通常のゲーム：敵を倒す → 満足感
このゲーム：人を救う → 使命感・達成感
```

その違いが、全てのシステムに反映されている。
