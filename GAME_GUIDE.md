# 戦場の衛生兵 - ゲームガイド

## 🎮 ゲーム概要

**タイトル**: 戦場の衛生兵（Battlefield Medic）  
**ジャンル**: スマートフォン向け縦画面ラン&回避アクション  
**対応**: Android / iPhone（縦画面専用）

---

## 📖 ストーリー・コンセプト

### コンセプト
**「武器を持たず、人を助ける」**

プレイヤーは戦場の衛生兵となり、銃弾や砲撃が飛び交う危険地帯を駆け抜けながら負傷兵を救助し、ヘリコプターまで搬送する。敵を倒すゲームではなく、**人命救助に特化したアクションゲーム**。

ハクソー・リッジに登場する衛生兵のような体験を目指す。

### プレイヤーに与えたい感情
- 🔴 **怖い** - 砲撃が迫ってくる緊張感
- ⏰ **間に合うか** - 時間制限による焦燥感
- 💚 **助けたい** - 負傷兵を救助する使命感
- 😤 **あと一人だけ助けたい** - スコア欲
- 🏃 **生きて帰りたい** - サバイバル感覚
- 🎉 **達成感** - 人命救助を成功させた満足感

---

## 🕹️ ゲームの流れ

### ステージ進行
1. **ステージ開始** - ミッションブリーフィング表示
2. **自動前進** - キャラが走り続ける（スクロール）
3. **障害物回避** - 砲撃・機銃掃射・有刺鉄線などを避ける
4. **負傷兵救助** - ランダム配置された負傷兵に接触して救助
5. **ヘリコプター到達** - ゴール地点に到達
6. **スコア獲得** - 救助人数に応じてスコア加算
7. **次のステージへ** - ステージ2→3→ミッション完了

## 🎯 ゲームシステム

### 基本操作
| 操作 | 入力 | 説明 |
|------|------|------|
| **ジャンプ** | 画面タップ | 上に飛ぶ（1回） |
| **二段ジャンプ** | ブーツ取得後、空中でタップ | ブーツ取得時のみ可能 |
| **スライディング** | 下スワイプ | 下に身を低くする（未実装） |

### ライフシステム
- **初期HP**: 5ポイント
- **ダメージ受けた場合**: -1 HP
- **無敵時間**: ダメージ後30フレーム無敵状態（点滅表示）
- **HP 0**: ゲームオーバー
- **ブーツ喪失**: ダメージを受けるとブーツが壊れる（二段ジャンプ不可になる）

## 🎁 アイテムシステム

### ブーツ（Boots）
| 項目 | 詳細 |
|------|------|
| **見た目** | 黄色とオレンジ色の靴型アイコン（16x16px） |
| **出現条件** | **ランダムに280フレーム（約4.7秒）ごと、50%確率で出現** |
| **出現位置** | スクロール位置の奥（敵と同じく上から下へ）、ランダムX座標 |
| **取得方法** | キャラが接触すると自動取得 |
| **効果** | **二段ジャンプが解放される** |
| **ジャンプ力** | 標準ジャンプと同じ力（11パワー） |
| **喪失条件** | **ダメージを受けると即座に喪失** |
| **連続取得** | 可能（複数回出現すれば何度でも取得可） |
| **戦略的価値** | 難易度の高いギミック回避に必須 |

### ブーツ取得時の能力変化
```
【ブーツなし】
- 地上ジャンプ：可（1回）
- 空中ジャンプ：不可
- 最大高さ到達時間：短い

【ブーツ取得後】
- 地上ジャンプ：可（1回）
- 空中ジャンプ：可（1回）＝計2段ジャンプ
- 複雑な障害物回避が可能
- より高い位置に到達可能
```

### ブーツの利用テクニック
1. **砲撃を高くジャンプで回避** - 標準ジャンプだけでは躱しきれない配置に有効
2. **機銃掃射の段階的躱し** - 1段ジャンプで低く躱す→2段目で高く上昇
3. **落石との立体的回避** - 3ステージの落石は2段ジャンプ必須
4. **ブーツ温存戦略** - 危険な場面に備えて、使用機会を選定
5. **連鎖取得** - ブーツを取得→失わずに次のブーツを獲得できれば常時二段ジャンプ可能

## 📊 アイテム出現データ

### ブーツ出現頻度と確率
| ステージ | 出現間隔 | 出現確率 | 1ステージ期待数 |
|---------|--------|--------|-----------|
| **1** | 280F | 50% | 約3～4個 |
| **2** | 280F | 50% | 約3～4個 |
| **3** | 280F | 50% | 約3～4個 |

※ 1ステージ＝約300秒＝18000フレーム

### 推奨ブーツ取得タイミング
- **ステージ1**: ブーツはボーナス程度。取得できれば得
- **ステージ2**: 機銃掃射出現後は二段ジャンプが便利
- **ステージ3**: **ブーツ必須エリア**（有刺鉄線＋落石組み合わせ）

### スコアシステム
| アクション | スコア | 説明 |
|----------|--------|------|
| **負傷兵1人救助** | **+100点** | スコアの主要な稼ぎ源。合計で500～1800点の獲得が可能 |
| **ステージクリア** | 救助人数 × 100 | ゲーム終了時の最終スコアは救助人数のみで決定 |
| **ゲームオーバー** | スコア保存 | 途中終了でもローカルストレージに記録 |
| **ハイスコア更新** | 自動保存 | localStorage に `bf_high` として保存 |
| **総救助人数** | 累計記録 | localStorage に `bf_total` として全プレイ合計を保存 |

#### 最大スコア理論値
- **理論最高**: 全ステージで全負傷兵救助
- **ステージ1**: 最大4人 × 100 = 400点
- **ステージ2**: 最大5人 × 100 = 500点
- **ステージ3**: 最大6人 × 100 = 600点
- **総合最高**: 1500点（3ステージすべてクリア＋全救助）

## 👤 キャラクター情報

### 衛生兵（プレイヤー）
- **見た目**: 8x12ピクセルのレトロドット絵
- **装備**: 緑色ユニフォーム + 赤十字マーク
- **特徴**: 走行アニメーション付き
- **HP表示**: キャラ上部に赤いピクセルでHP表示
- **能力**:
  - 標準ジャンプ（11パワー）
  - ブーツ取得時に二段ジャンプ可能
  - ダメージ受けると30フレーム無敵状態

### 負傷兵
- **見た目**: 肌色の頭 + 赤い体 + 黒い脚（ドット絵、20x20px）
- **配置**: スクロール位置の奥、ランダムX座標に出現
- **救助方法**: キャラが接触すると自動救助（接触時に青緑色のパーティクル発生＋成功音）
- **スコア**: 1人につき100点
- **出現頻度**:
  - **ステージ1**: 150-15=135フレーム毎（約2.25秒毎）
  - **ステージ2**: 150-30=120フレーム毎（2秒毎）
  - **ステージ3**: 150-45=105フレーム毎（約1.75秒毎）
- **期待救助人数**: ステージ1で3～4人、ステージ2で4～5人、ステージ3で5～6人

## ⚠️ ギミック詳細

### 共通仕様
- **出現方式**: すべてのギミックは奥から手前に流れてくる（スクロール）
- **当たり判定**: 矩形判定（ピクセルパーフェクト）
- **ダメージ**: どのギミックも接触で-1 HP + ブーツ喪失
- **無敵時間**: ダメージ後30フレーム（その間は再接触しない）

### ステージ1: 海岸戦線（難易度★☆☆）
**主要ギミック**: 砲撃

| ギミック | 見た目 | 出現条件 | 説明 |
|---------|--------|--------|------|
| **砲撃爆発** | オレンジ→黄→黄 3段構造 | 70F毎に60%確率 | 高速で降ってくる。ジャンプで回避。サイズ50x50px |

**特徴**: 砲撃のみで敵対的。初心者向けステージ。ギミック数少ないため学習用。

---

### ステージ2: 塹壕地帯（難易度★★☆）
**主要ギミック**: 砲撃 + 機銃掃射

| ギミック | 見た目 | 出現条件 | 説明 |
|---------|--------|--------|------|
| **砲撃爆発** | オレンジ→黄→黄 3段構造 | 70F毎に60%確率 | ステージ1と同じ。サイズ50x50px |
| **機銃掃射** | 黄色＋オレンジ横条（8本） | 70F毎に20%確率 | 画面全幅を占める。縦幅20px。躱しづらい |

**特徴**: 複数ギミックの同時出現あり。避けるパターンが増える。機銃は身を低くしないと躱しきれない。

---

### ステージ3: 廃墟の村（難易度★★★）
**主要ギミック**: 砲撃 + 機銃掃射 + 有刺鉄線 + 落石

| ギミック | 見た目 | 出現条件 | 説明 |
|---------|--------|--------|------|
| **砲撃爆発** | オレンジ→黄→黄 3段構造 | 70F毎に60%確率 | 出現頻度最高。サイズ50x50px |
| **機銃掃射** | 黄色＋オレンジ横条（8本） | 70F毎に20%確率 | 画面全幅。更に頻繁に出現 |
| **有刺鉄線** | マゼンタ（紫）＋濃いマゼンタ | 70F毎に20%確率 | ステージ3のみ登場。地上に配置。サイズ40x40px |
| **落石** | 実装予定（現在は有刺鉄線でカバー） | - | 上から落ちてくる障害物（将来実装） |

**特徴**: 全てのギミックが登場。制限時間260秒と短い。有刺鉄線の地上設置で高くジャンプ必須＝ブーツ需要高い。

## 🎵 サウンド

### 効果音（Web Audio API生成）
| イベント | 周波数 | 長さ | 特徴 | 説明 |
|---------|-------|------|------|------|
| **通常ジャンプ** | 400Hz | 0.1秒 | 上昇音 | プレイヤーが地面からジャンプした時の音 |
| **二段ジャンプ** | 500Hz | 0.1秒 | より高い上昇音 | ブーツ取得時の空中ジャンプ音 |
| **ダメージ受けた** | 100Hz | 0.2秒 | 低い爆発音 | ギミック接触時の被ダメージ音 |
| **負傷兵救助** | 800Hz | 0.3秒 | 高い成功音 | 負傷兵に接触した時の救助完了音 |

### 音声フィードバック機能
- **指数減衰**: すべての効果音は徐々に音量が下がる（exponentialRampToValueAtTime）
- **発生トリガー**: 画面に見えている/いないに関わらず発生
- **Web Audio API**: 最新ブラウザで低遅延音声出力
- **フォールバック**: 非対応ブラウザでは音声なし（エラー処理で無視）

### BGM
- **ステージ1-3**: 緊迫した戦場音楽（**将来実装予定**）
- **現在**: 効果音のみで臨場感表現

## 🎮 プレイテクニック

### 基本戦略
1. **標準ジャンプで砲撃回避** - 地上の砲撃は飛び越える
2. **二段ジャンプはボス/難敵用に温存** - ブーツを失わないよう注意
3. **機銃掃射は避ける** - 高さを変えて躱す
4. **負傷兵は積極的に救助** - スコアと使命感を両立

### 上級テクニック
- ブーツ取得直後に積極的に二段ジャンプ活用
- ギミック出現パターンを予測して事前ジャンプ
- スコア狙いプレイ（できるだけ多くの負傷兵を救助）
